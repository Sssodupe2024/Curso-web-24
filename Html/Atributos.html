Atributos en elementos de HTML

Todos los elementos pueden contener atributos. Los atributos proveen Información adicional sobre los elementos.

Los atributps se especifican en la etiqueta de apertura. Se componen de dos partes, nombre y valor.

name= "value"

href

Es el atributo básico de los links (a). Las (a) define un hipervínculo, el atributo href define la URL de la página web dónde nos va a llevar el hiprvínculo.

src

La etiqueta ( ing) se usa para insertar una imagen en una página HTML. El atributo src indica el camino o dirección de la imagen a mostrar.

Width y height

Los atributos width, ancho y height, altura, definen el tamaño que queremos otorgar a la imagen de nuestra página.

Alt

El atributo alt define el texto alternativo para una imagen. Este texto se mostrara si por alguna razon la imagen no puede cargarse.

Style

Este atributo se usa para añadir estilos visuales a cualquier elemento. Como por ejemplo, color, fuente-font, tamaño , size etc.
