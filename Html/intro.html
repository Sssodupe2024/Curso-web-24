Introducción a HTML

¿Qué es HTML?

HTML significa Hyper Text Markup Language. En castellano, lengua de marca de hipertexto.

Html describe la estructura de una página web. Consiste en una serie de elementos que indican al navegador como mostrar la página.

Los elemenntos marcan los trozoz de contenido y lo describen como esto es un párrafo, esto es un titulo", " esto es una imagen/ link/ botón"...

¿Qué es un elemento?

Estructura de un elemento

Contenido

Un elemento es todo aquello desde la apertura hasta la etiqueta de cierre. Casi todos los elementos estan formados por etiquetas de apertura y cierre, salvo excepciones.

Ejemplo:
La etiqueta "br" es una etiqueta vacía, es decir, no tiene contenido ni etiqueta de cierre. Sirve para indicar un salto de línea.
